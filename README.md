# 🤖 باكي ستور — البوت الإداري

بوت تيليغرام إداري خاص بيك تضيف وتحذف منتجات وكوبونات مباشرة من تيليغرام.

---

## 📋 خطوات التشغيل

### الخطوة 1: إنشاء بوت إداري جديد
1. افتح تيليغرام وتواصل مع [@BotFather](https://t.me/BotFather)
2. أرسل `/newbot`
3. اكتب اسم البوت مثل: `Baki Admin Bot`
4. اكتب username مثل: `baki_admin_bot`
5. احتفظ بالـ **TOKEN** اللي يعطيك إياه

### الخطوة 2: احصل على ADMIN_CHAT_ID
- هذا هو chat_id الخاص بيك (مو chat_id بوت الطلبات)
- تواصل مع [@userinfobot](https://t.me/userinfobot) وراح يعطيك ID حسابك

### الخطوة 3: أضف الملفات لمشروع Vercel
- ضع مجلد `api/` داخل مشروعك الحالي على GitHub
- الملف: `api/bot.js`
- الملف: `vercel.json`

### الخطوة 4: أضف Environment Variables في Vercel
اذهب إلى مشروعك في Vercel → Settings → Environment Variables:

| الاسم | القيمة |
|-------|--------|
| `ADMIN_BOT_TOKEN` | TOKEN البوت الإداري الجديد |
| `ADMIN_CHAT_ID` | chat_id حسابك في تيليغرام |

### الخطوة 5: سجّل الـ Webhook
بعد النشر على Vercel، افتح المتصفح واذهب إلى:
```
https://api.telegram.org/botYOUR_TOKEN/setWebhook?url=https://your-project.vercel.app/api/bot
```
أو شغّل `setup-webhook.js` بعد تعديله.

## 🎮 أوامر البوت

| الأمر | الوظيفة |
|-------|--------|
| `/start` أو `/menu` | القائمة الرئيسية |
| `/cancel` | إلغاء العملية الحالية |
| زر ➕ إضافة منتج | يبدأ خطوات إضافة منتج |
| زر 📦 عرض المنتجات | يعرض كل المنتجات مع ID كل واحد |
| زر 🗑 حذف منتج | يطلب ID المنتج ويحذفه |
| زر 🎟 إضافة كوبون | يضيف كوبون خصم جديد |
| زر 📊 إحصائيات | يعرض إحصائيات المتجر |

## 🔒 الأمان
- البوت يقبل فقط رسائل من ADMIN_CHAT_ID الخاص بيك
- أي شخص ثاني يحاول يستخدمه يحصل على رسالة رفض
